# Ask Mode Rules (Non-Obvious Only)

- Project uses multiple web frameworks: Actix-web (primary), Spring-web (secondary), Axum (minimal examples)
- Database layer is SQLite with custom RBatis integration via w-ddd and w-macro crates
- Domain-driven design patterns implemented in w-ddd with custom procedural macros
- Blog application has three entry modes: web server (run), simple greeting (hello), alternative greeting (hello2)
- Algorithm crate contains educational examples of divide & conquer, search, and sort algorithms
- Chinese chess game is a desktop GUI application using eframe/egui framework
- Web utilities (w-*) crates serve as examples and shared infrastructure for web applications
- Configuration system uses global static with thread-safe initialization guards
- Procedural macros generate entity repositories with automatic CRUD operations
